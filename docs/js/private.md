- 保密性
- 数据完整性
- 身份校验

分类

- 对称加密

DSA 数字签名

AES 高级加密标准，对称算法 128/256


- 非对称加密
加密解密使用不同的密钥
公钥（对外公布）和私钥（不对外公布）

S={P, C, K, E, D}

RSA 数论非对称性(公开钥)加密算法

常采用传统加密方法与公开密钥加密方法相结合的方式，即信息采用改进的DES或IDEA对话密钥加密
然后使用RSA密钥加密对话密钥和信息摘要
